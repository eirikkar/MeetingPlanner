# Meeting Planner

## Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/eirikkar/MeetingPlanner.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd MeetingPlanner
    ```
3.  Install the dependencies:
    ```sh
    dotnet restore
    ```

## Usage

1.  Start the program:
    ```sh
    dotnet run
    ```
2.  The application will start in the terminal.
